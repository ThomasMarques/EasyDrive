Today there are a lot of possibility to upload en share files, but you always have to use a big and heavy user interface. With a simple terminal EasyDrive will save your time.
easydrive - term
Welcome to EasyDrive Terminal, type help to see command.

Server commands

-cd `relative or absolute path.
-ls
-pwd
-mkdir
-rm `file or directory.
-cp (TODO)
-mv
-download / get `file or directory(zip)`
 -find / search
Usage : `find (or search) nameToSearch` to search in the current directory, `find (or search) nameToSearch -a` to search in your all content, `find (or search) nameToSearch directoryToSearch` to search in a specific directory.
-share / chmod
Usage : `share (or chmod) FileOrDirectoryName userLogin`.
